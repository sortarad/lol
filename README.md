## Installation

Install via the Control Panel or via composer

```bash
composer require sortarad/lol
```

## Enable Widget

Add widget to control panel configuration `config/statamic/cp.php` file.

e.g:
```php
[
	'type' => 'lol',
	'width' => 50,
],
```
